Basic HTML tags

This is a basic html tags that is commonly used. 

<div></div> --> is called Division, used to contain your web content in here 

<h1></h1>, <h2></h2>, <h3></h3>, <h4></h4>, <h5></h5> --> this is a heading tags. 1 is biggest font and 5 is smallest font

<p></p> --> This is paragraph tags used when u want to type long sentence in html

<ul> --> Unordered List Tags. This is a list that is list unorderedly. This will display as bulletpoint

	<li></li> --> this a first item in the list using li tag

	<li></li> --> this a second item in the list using li tag

</ul>

<ol> --> Ordered List Tags. This is a list that is list Orderly. This will display as 1, 2 and so on

	<li></li>--> this a first item in the list using li tag

	<li></li> --> this a second item in the list using li tag

</ol>

<button></button> --> This is a button tag used as .. well a button

<form></form> --> This is a form tag used to submit data ~

<img>--> image tag used for IMAGE ! duh ~



bonus HTML5 tags

<nav></nav> --> nav used for navigation bar

<aside></aside> --> aside used for Sidebar normally

<main> --> main is where ur main content of the site will be

	<section></section> --> section is the section of your content

</main>

<footer></footer> --> footer of your site

Layout

  <title></title>
  <style type="text/css"></style>
</head>
<body>
  <!-- This is for header like navbar -->
  <header>
  </header>
  
  <!-- This is for Sidebar -->
  <aside>

  <!-- This is for the main content of the web page -->
  <main>
    <!-- This is for the section of the content-->

      <div>
        <form>

    </section>

</html>

Row stands for Row(duh ~) and Col stands for column. The idea is, you seperate the screen into multiple row and column, specifically 12 column. You can have unlimited rows tho(not that you should~)
